# Spotify
Trying to build a basic Frontend website with the help of HTML and CSS to learn and to implement the concepts on ground.

## About the project
This project is the step by step implementation of music streaming UI. By building this project i am gaining hands on experience on how real world designs are implemented by adding core structure through HTML and styling through CSS.

## Tech Stack
**HTML 5** - Structuring the project

**CSS** - Styling the project

**Font-awesome,google fonts** - for adding the styling components


## Features implemented
 **Sidebar Navigation**: Home, Search, and Library sections.

 **Sticky Navigation Bar**: Top navigation with user profile and upgrade buttons.

 **Main Content Area**: Product cards for "Recently Played" and "Spotify Original Podcasts".

 **Music Player**: Bottom player control bar featuring song details, playback progress sliders, and volume adjustments.

 ## Difficulties Faced
  **Overflow property**: Not able to  scroll the entire content on the main section then implemenetd overflow property.
  
  **Responsiveness**: Intially facing issue to implement flex properties and use relative units for styling to make website responsive in nature.

 ## Learning
  Managing the UI components properly how it is divided into chunks and then build.
